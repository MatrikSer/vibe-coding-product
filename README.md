# Угадай число (консольная игра)

Простая консольная игра: компьютер загадывает случайное число от 1 до 100, а вы пытаетесь угадать. После каждой попытки игра подсказывает, нужно ли число **больше** или **меньше**. Когда число угадано, выводится количество попыток.

## Запуск

- **Требования**: Python 3.8+ (подойдёт любой современный Python 3)
- **Запуск игры**:

```bash
python main.py
```

## Правила

- Введите целое число.
- Игра подскажет: `Больше` или `Меньше`.
- При угадывании вы увидите поздравление и число попыток.
